# Functions-and-Conditional-Logics-Project
Layihə Haqqında
Bu repozitoriyada Python-da Şərtli Məntiqlər (Conditional Logics) və Funksiyalar (Functions) mövzuları üzrə işlədiyim praktiki tapşırıqlar yer alır. Əsas məqsəd mürəkkəb məntiqləri təmiz və təkrar istifadə edilə bilən kod halına gətirməkdir.

Nələri Öyrəndim və Tətbiq Etdim?
if-elif-else şərtləri: Müxtəlif ssenarilərə uyğun qərarvermə alqoritmlərinin qurulması.

Funksiyalar (def): Kodun təkrarçılığının qarşısını almaq və daha oxunqlu struktur yaratmaq.

Məntiqi Analiz: Biznes tapşırıqlarını Python koduna çevirmək bacarığı.
